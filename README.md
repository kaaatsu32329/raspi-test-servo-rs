## Install Rust on Raspberry Pi

Reference: [rust-lang](https://www.rust-lang.org/tools/install)

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

## Check your pin assign

```bash
pinout
```

## Build and Run

```bash
cargo run
```
